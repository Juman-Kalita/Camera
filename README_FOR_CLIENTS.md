# 🎯 Quick Start for Clients

## Super Simple Setup (5 Minutes)

### For Windows Users:

1. **Download the code:**
   - Go to: https://github.com/Juman-Kalita/Camera
   - Click green "Code" button → "Download ZIP"
   - Extract the ZIP file

2. **Run the installer:**
   - Double-click `EASY_INSTALL.bat`
   - Wait 2-3 minutes for installation
   - Done!

3. **Start using:**
   - Double-click "Workplace Analytics" on your desktop
   - Dashboard opens automatically
   - System starts monitoring

### For Mac/Linux Users:

1. **Download the code:**
   ```bash
   git clone https://github.com/Juman-Kalita/Camera.git
   cd Camera
   ```

2. **Install:**
   ```bash
   pip3 install -r requirements.txt
   ```

3. **Run:**
   ```bash
   chmod +x start.sh
   ./start.sh
   ```
